10. Declarar e inivializar una variable de tipo array de Object de longitud 3. Inivializar el primer elemento del array con un array de ints, el segunfo con un array de arrays de ints y el tercero con un array de Objects. Observar si alguna de las asignaciones falla al compilar.

R: Las tres compilan y se ejecutan de sin problemas, debido a que los arrays son todos descendientes de la clase Object y los arrays son covariantes, lo que significa que se puede almacenar sin problemas elementos de las clases hijas de la clase con la que se declaró el array. Por lo tanto, es posible asignar cualquier array a cualquier variable de tipo Object (aunque sea miembro de un array). El problema llegará cuando se intente acceder a los elementos de los arrays internos (ver y ejecutar líneas comentadas en el ejemplo). La JVM detectará en qué momento se tiene un Object en lugar de un array y no permitirá agregar más corchetes de los declarados originalmente. Para acceder a un array, debe haber sido declarado como array (no como Object)
